# Ragdoll-torture
Have you ever wanted to just throw a bomb at someone you know? 
Now you can with Torture Simulator
## Features
* Realistic Ragdoll Physics 😦
* 2 sizes of Bombs 💣
* Keyboard Support ⌨️
* Spawnable Dummies 👀
* Dummie Deaths 😵
* Custom Dummies😎
* Playable in any webgl browser 💻
